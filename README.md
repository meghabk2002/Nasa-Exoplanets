# 🌌 NASA Exoplanet Discovery Analysis (Power BI Dashboard)

## 📌 Project Overview

This project explores **exoplanet discoveries using NASA’s dataset** and visualizes how planets outside our solar system have been discovered over time. The dashboard highlights discovery trends, telescope contributions, planet types, and the possibility of **habitable planets similar to Earth**.

Using **Power BI**, the data is transformed into an interactive dashboard that allows users to explore planetary discoveries through filters and dynamic visuals.

---

## 🎯 Objectives

The goal of this project is to:

* Analyze **exoplanet discovery trends over time**
* Identify **which telescopes discovered the most planets**
* Understand **different planet types and their characteristics**
* Explore **potentially habitable planets**
* Build an **interactive Power BI dashboard for data storytelling**

## 📊 Dashboard Pages

### 1️⃣ Discovery Era

This page focuses on how exoplanet discoveries have evolved over time.

Key insights include:

* Total number of planets discovered
* Number of habitable planets
* Discovery trends by year
* Most used discovery methods
* Telescopes responsible for the majority of discoveries

Key Visuals:

* Discovery timeline (line chart)
* Discovery method distribution (donut chart)
* Telescope contribution (bar chart)
* Interactive filters for discovery method and year

---

### 2️⃣ The Planets

This page explores the **physical characteristics of discovered planets**.

Key insights include:

* Distribution of different planet types
* Relationship between **planet size and orbital period**
* Average radius of planet categories
* How planet discoveries evolved over time by type

Key Visuals:

* Scatter plot: Planet size vs orbital period
* Planet type distribution (donut chart)
* Average planet size by type (bar chart)
* Planet discoveries over time (stacked chart)

Planet categories analyzed:

* Gas Giants
* Neptune-like
* Super Earth
* Terrestrial

---

### 3️⃣ Another Earth?

This page investigates the **possibility of habitable planets**.

Key insights include:

* Number of planets that could potentially support life
* Influence of planetary properties on temperature
* Analysis of radius, mass, and temperature ranges
* Identification of potentially habitable planets

Key Visuals:

* Key Influencers visual (Power BI AI feature)
* Habitable planet KPI
* Interactive filters for:

  * Planet radius
  * Temperature
  * Telescope
* Table showing potentially habitable planets

---

## 📈 Key Insights

* Over **6000+ exoplanets** have been discovered.
* The **Transit method** accounts for the majority of discoveries.
* The **Kepler Telescope** discovered the largest number of exoplanets.
* **Neptune-like planets** are the most commonly discovered planet type.
* A small percentage of discovered planets fall within the **potentially habitable zone**.
